# 🏥 Sistema de Triagem Hospitalar

Projeto desenvolvido em C que simula o funcionamento de uma triagem em pronto atendimento, utilizando estruturas de dados para organizar pacientes por nível de prioridade.

---

## ⚙️ Funcionamento

O sistema organiza os pacientes em filas de acordo com o nível de gravidade:

- 🔴 Vermelho — atendimento imediato  
- 🟡 Amarelo — urgência moderada  
- 🟢 Verde — menor urgência  
- 🔵 Azul — casos simples  

O atendimento segue:
- prioridade de gravidade  
- ordem de chegada dentro de cada fila (FIFO)

## 🧠 Conceitos aplicados

- Estruturas de dados (filas)
